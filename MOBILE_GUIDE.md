# 📱 スマホでの使用方法

このアプリはスマホでも完全に動作するようにデザインされています。

## 🚀 クイックスタート

### 1. アプリを開く

ブラウザで以下のURLにアクセス:
```
https://your-deployed-url.vercel.app
```

### 2. ホーム画面に追加（推奨）

**iPhone/iPad:**
1. Safari を開く
2. 共有ボタン（↑） をタップ
3. "ホーム画面に追加" を選択
4. "追加" をタップ

**Android:**
1. Chrome/Edge を開く
2. メニュー（⋮） をタップ
3. "アプリをインストール" を選択
4. "インストール" をタップ

### 3. アプリを使う

- ホーム画面のアイコンからアプリを起動
- ネイティブアプリと同じように動作
- インターネット接続がなくても使用可能

## 📊 主な機能

### 🏋️ ワークアウト追跡
- **新規ワークアウト**: 下のナビから "New Workout"
- **エクササイズ追加**: 筋肉グループを選択して追加
- **タイマー**: 自動的に表示される（バックグラウンドで動作）

### 📈 進捗確認
- **ホーム**: カレンダーで最近のワークアウトを確認
- **Progress**: グラフで進捗を視覚化
- **History**: 過去のワークアウト記録

### 📊 健康追跡
- **Measurements**: 体重・体脂肪などを記録
- **Nutrition**: 食事とマクロ栄養を追跡
- **Goals**: 目標を設定

## 🔔 タイマー機能

### 使用方法
1. エクササイズを追加
2. 自動的にタイマーが画面右下に表示
3. タイマーが 0:00 になってもその後の超過時間を表示
4. "✕" をクリックして終了

### カスタマイズ
- ホーム画面の ⚙️ Settings でサウンド/バイブレーション設定
- バックグラウンドで動作（アプリを最小化しても継続）

## 💾 データ管理

### 自動保存
- すべてのデータは自動的に保存
- インターネット接続不要
- デバイスの内部ストレージに保存

### バックアップ/復元

**エクスポート:**
1. "Sync" をタップ
2. "Export Data" をクリック
3. JSONファイルをダウンロード

**インポート:**
1. 別のデバイスで "Sync" を開く
2. "Import Data" をクリック
3. ダウンロードしたJSONを選択

## ⚙️ 設定

### タイマー設定
- ホーム画面 → ⚙️ Settings
- サウンド有/無
- バイブレーション有/無

### マニフェスト設定
- アプリ名やアイコンをカスタマイズ可能
- `public/manifest.json` を編集

## 🐛 トラブルシューティング

### アプリが遅い
- キャッシュをクリア
- アプリを再起動

### データが表示されない
- インターネット接続を確認
- ブラウザの Local Storage を確認

### タイマーが始まらない
- ブラウザをリロード
- キャッシュをクリア

### インストール画面が表示されない
- HTTPSサイトであることを確認
- ブラウザの最新版を使用
- サードパーティCookieを有効にする

## 📲 推奨環境

**推奨:**
- iPhone 12 以上 + Safari
- Android 8 以上 + Chrome/Edge

**最小要件:**
- タッチスクリーン搭載
- 100MB 空き容量
- 高速インターネット接続（初回ダウンロードのみ）

## 💡 ヒント

1. **ホーム画面に追加**: ブラウザのブックマークより便利
2. **オフラインで使用**: インターネット接続なしで完全に動作
3. **データシェア**: 友人と JSON ファイルを共有してデータを共有
4. **背景のままにする**: アプリを最小化してもタイマーが続行
