# 🚀 クイックスタート - GitHub Desktopでアップロード

## ⚡ 5分で完了！GitHub Desktopアップロード手順

### 📋 準備物
- ✅ GitHubアカウント
- ✅ GitHub Desktop (インストール済み)
- ✅ このプロジェクトフォルダ

---

## ステップ1: GitHub.comでリポジトリ作成 (1分)

1. **ブラウザでGitHubを開く**
   ```
   https://github.com
   ```

2. **新しいリポジトリ作成**
   - 右上の `+` → `New repository`
   - **名前**: `workout-tracker`
   - **説明**: `Mobile fitness tracker app`
   - **Public** を選択
   - **READMEファイル追加** のチェックを外す
   - **Create repository** をクリック

3. **URLをコピー**
   - 緑の "Code" ボタン → URLをコピー
   - 例: `https://github.com/yourusername/workout-tracker.git`

---

## ステップ2: GitHub Desktopでプロジェクト追加 (2分)

1. **GitHub Desktop起動**

2. **プロジェクトフォルダを開く**
   - `File` → `Add local repository...`
   - `Choose...` をクリック
   - フォルダ選択: `C:\Users\user\.vscode\workout-tracker-main`
   - `Add repository` をクリック

3. **GitHubに接続**
   - "Publish repository" をクリック
   - **名前**: `workout-tracker` (自動入力)
   - **説明**: `Mobile fitness tracker app`
   - **公開** にチェック
   - **Publish repository** をクリック

---

## ステップ3: ファイルをアップロード (2分)

1. **コミット作成**
   - コミットメッセージ: `Initial commit - Mobile fitness tracker`
   - `Commit to main` をクリック

2. **GitHubにプッシュ**
   - `Push origin` をクリック
   - アップロード完了まで待つ

---

## ✅ 完了確認

1. **GitHub.comで確認**
   - リポジトリページを開く
   - ファイルが表示されているか確認

2. **GitHub Desktopで確認**
   - "Fetch origin" をクリック
   - 同期されていることを確認

---

## 🎯 次は何をする？

### Vercelでデプロイ (推奨)
1. [vercel.com](https://vercel.com) にアクセス
2. GitHubアカウントでログイン
3. リポジトリをインポート

### スマホでテスト

2. ホーム画面に追加 (PWAインストール)
3. ネイティブアプリのように使用

---

## 🆘 問題が発生したら

### GitHub Desktopが起動しない
- 再インストールしてみる
- Windowsを再起動

### リポジトリが見つからない
- GitHub.comでリポジトリが作成されているか確認
- URLが正しくコピーされているか確認

### プッシュが失敗する
- インターネット接続を確認
- GitHub Desktopでサインインし直す

### 詳細ガイド
- **[完全版ガイド](GITHUB_DESKTOP_GUIDE.md)** を参照
